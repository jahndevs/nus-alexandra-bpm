# Integrating Cuffless Blood Pressure Monitoring at Alexandra Hospital

## Overview
Designing and prototyping a non-invasive cuffless blood pressure monitor for use in Alexandra Hospital that is clinically accurate. As part of that process, our wearable will use PPG signals measured on the wrist which is then processed and passed through a machine learning algorithm to predict blood pressure readings. This project is in collaboration with the National University of Singapore and Alexandra Hospital.

## Web Dashboard
Uses a React/Vite frontend using MUI components with a python backend (FastAPI) for fast transmission of real-time data from wrist wearable device to dashboard.
